# Code of Conduct

This project adopts the [Contributor Covenant v2.1][cc] as its code of
conduct. The full text is canonical; please read it there rather than
relying on a local copy that could drift.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

The Covenant applies to all project spaces — issues, pull requests,
discussions, commit messages, and any public communication that
represents the project.

## Reporting

Conduct concerns may be reported privately to the maintainer
([`@jkindrix`](https://github.com/jkindrix)) via a GitHub security
advisory or by emailing the address listed on the maintainer's GitHub
profile. Reports will be acknowledged within 72 hours. Reporters'
identities will be kept confidential to the extent the maintainer can
do so while still acting on the report.

## Enforcement

The maintainer is responsible for clarifying and enforcing the
Covenant's standards. Consequences for confirmed violations follow the
Covenant's
[Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines):
correction, warning, temporary ban, permanent ban. The maintainer
reserves the right to skip rungs for sufficiently severe behavior.
